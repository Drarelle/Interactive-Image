Interactive Image
=================

A jQuery plugin to embed interactive images on your website.

`Online Demo <https://www.jpchateau.com/demo/interactive-image>`_.

Requirements
------------

`jQuery`_ 1.7.2+ is required.

Dependencies
------------

-  `imagesloaded`_ to detect when images have been loaded
